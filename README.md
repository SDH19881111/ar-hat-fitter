# 글로벌 전통 모자 AR 피팅룸 (AR Hat Fitting Room)

이 프로젝트는 브라우저와 웹캠을 연동하여 세계 여러 나라의 전통 모자를 실시간 AR로 착용해보고 사진을 찍을 수 있는 웹 애플리케이션 프로토타입입니다.

## 적용된 기술
*   **MediaPipe Face Landmarker**: 서버 없이 순수 클라이언트 사이드 브라우저 환경에서 실시간 60fps로 얼굴 랜드마크(468개 좌표)를 추적합니다.
*   **Chroma Key Background Removal**: 생성형 AI로 제작된 흰색 배경의 모자 이미지들을 로딩 시 실시간으로 배경 투명화 처리(누끼 따기)를 진행하여 웹캠 화면에 깔끔하게 합성합니다.
*   **Canvas Transformation**: 얼굴의 크기(가로/세로) 및 기울기(Roll 각도)를 실시간 계산하여 모자의 크기 및 각도를 자연스럽게 매핑합니다.
*   **Premium Web Design**: 다크 모드 기반의 글래스모피즘(Glassmorphism) 테마와 네온 그래디언트 디자인, 사진 촬영 시 플래시 효과 등을 지원합니다.

## 포함된 전통 모자 컬렉션
1.  **인디언 모자** (북아메리카 전통 깃털 머리띠)
2.  **카우보이 모자** (미국 서부 가죽 모자)
3.  **베트남 논라** (베트남 전통 대나무 모자)
4.  **파라오 왕관** (고대 이집트 네메스)

## 실행 방법

로컬 웹 서버를 구동하여 이 폴더의 `index.html`을 브라우저에 띄우면 바로 실행할 수 있습니다. (MediaPipe 모델 로딩 및 카메라 작동을 위해 로컬 파일 더블클릭이 아닌 **로컬 서버 구동**이 필수입니다.)

이 폴더(`C:\Users\user\.gemini\antigravity\scratch\ar-hat-fitter`)에서 아래 명령어를 실행하여 로컬 서버를 열 수 있습니다.

```bash
# npx를 이용하여 임시 웹 서버 열기
npx -y http-server
```

서버가 실행되면 터미널에 나타나는 주소(예: `http://127.0.0.1:8080` 또는 `http://localhost:8080`)로 브라우저(Chrome 권장)에 접속하고, 카메라 권한을 허용하시면 실시간 AR을 경험하실 수 있습니다.
